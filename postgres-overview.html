PostgreSQL

  • PostgreSQL is a powerful open source object relation database system that uses and extend the SQL language
  • PostgreSQL has earned a strong reputation for its proven architecture, reliability, data intergrity, robust feature set, extensibility and the dedication of the open source community
  • PostgreSQL runs on all major operating systems and has been ACID- compliant since 2001
  • PostgreSQL have powerful addons such as the popular PostGIS geospatial database extender
